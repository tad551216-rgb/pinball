# つくる手帖ピンボール

思いつく → つくる → 届ける。アイデアの玉を弾いて遊ぶ、手帖のピンボールです。
バンパーは草の子（思）・もぐら（作）・手帖くん（届）。3つ全部ならすと「完成」、朱印が押されてボーナス。

- 1ファイル完結のゲーム本体に、PWA一式（マニフェスト・サービスワーカー・アイコン）を付けた「道具」版です。
- ローカル完結・通信なし・オフライン動作。ハイスコアは端末内（localStorage）にだけ保存します。
- ホーム画面に追加すると、他のつくる手帖アプリと同じく全画面で起動します。

## ファイル構成（リポジトリのルートに置く）

```
index.html                ゲーム本体（マスコット画像も埋め込み済み）
manifest.webmanifest      PWAマニフェスト
sw.js                     サービスワーカー（オフライン用）
icon-192.png              アイコン
icon-512.png              アイコン
icon-maskable-512.png     アイコン（マスカブル）
apple-touch-icon.png      iOSホーム画面用アイコン
```

## 公開方法（GitHub Pages）

1. 新しいリポジトリ `pinball` を作る（GitHubユーザー: tad551216-rgb）。
2. 上の7ファイルをすべて**ルート**に置く。
3. Settings → Pages → Deploy from a branch（main / root）で公開。
4. 公開URL: `https://tad551216-rgb.github.io/pinball/`

すべて相対パスなので、サブディレクトリ（プロジェクトページ）のままで動作します。

## つくる手帖（ポータル）の「道具」に追加する

ポータルの `index.html` 下部にある `APPS` 配列に、次の1件を足してください。
（※ 文字列の中で半角の `"` は使わず、『』を使っています。配列を壊さないため、この方針はそのままに。）

```js
{
  name:"つくる手帖ピンボール",
  status:"live",
  glyph:"🕹️",
  summary:"思いつく・つくる・届ける――アイデアの玉を弾いて遊ぶ手帖のピンボール。『思・作・届』を全部ならすとひとつ完成。ローカル完結・通信なし。",
  tags:["PWA","ゲーム","オフライン"],
  demo:"https://tad551216-rgb.github.io/pinball/",
  article:"#"
},
```

- `glyph` の絵文字や `tags` は好みで変えてください。
- 「作り方を読む」の記事をあとで書く場合は、`article` に `articles/pinball.html` などを指定します（他の記事と同じく `articles/kakeibo.html` をコピーして本文を差し替えると、見た目が自動でそろいます）。

## 操作

- PC: ← → （または A / L）でフリッパー、スペース長押し→はなして発射、Enter でリスタート。
- スマホ: 盤面の左半分／右半分タップでフリッパー、発射ボタン長押し→はなして発射。長押しの溜めで打ち出しの強さが変わります。
